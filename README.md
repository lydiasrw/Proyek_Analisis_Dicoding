✨ **Proyek Analisis Dicoding** ✨

Merupakan final test dari online course dicoding.com. Dataset berisi data e-commerce yang terdiri dari sembilan file. 
Analisa dilakukan menggunakan tahapan data wragling, menggunakan teknik analisis "Exploratory Data Analyst", dan visulisasi data dengan matplotlib.
Selanjutnya, semua data visualisasi dibuat dalam aplikasi streamlit
